🔥 Day 1 – Push (Chest, Shoulders, Triceps)
Chest (Primary Focus)
1️⃣ Smith Machine Flat Bench Press – 4 sets x 6-10 reps
2️⃣ Incline Machine Press (or Smith Machine Incline Press) – 3 sets x 8-12 reps
3️⃣ Dips Machine (Chest Focus, Lean Forward) – 3 sets x 8-12 reps
4️⃣ Cable or Machine Chest Flys – 3 sets x 12-15 reps

Shoulders (Secondary Focus)
5️⃣ Seated Machine Shoulder Press (or Smith Machine Shoulder Press) – 3 sets x 8-12 reps
6️⃣ Lateral Raises (Dumbbell or Cable) – 3 sets x 12-15 reps
7️⃣ Front Delts Machine – 3 sets x 12-15 reps

Triceps (Support & Isolation)
8️⃣ Close-Grip Bench Press (Smith Machine) – 3 sets x 8-12 reps
9️⃣ Overhead Machine/Dumbbell Triceps Extension – 3 sets x 10-12 reps
🔟 Cable Triceps Pushdowns – 3 sets x 12-15 reps

🔥 Day 2 – Pull (Back, Biceps, Rear Delts)
Back (Primary Focus)
1️⃣ Smith Machine Rack Pulls (or Deadlifts if preferred) – 4 sets x 5-8 reps
2️⃣ Pull-Ups (Assisted if needed) OR Lat Pulldown Machine – 4 sets x 6-10 reps
3️⃣ Seated Row Machine (Neutral or Wide Grip) – 3 sets x 8-12 reps
4️⃣ Lat Pulldown Machine (Close-Grip or Underhand Grip) – 3 sets x 10-12 reps

Rear Delts (Secondary Focus)
5️⃣ Rear Delt Flys (Dumbbell or Machine) – 3 sets x 12-15 reps
6️⃣ Face Pulls (Cable or Machine) – 3 sets x 12-15 reps

Biceps (Support & Isolation)
7️⃣ Machine Preacher Curls (or Dumbbell Curls) – 3 sets x 8-12 reps
8️⃣ Hammer Curls (Dumbbell or Rope Attachment on Cable) – 3 sets x 10-12 reps
9️⃣ Cable or Machine Bicep Curls – 3 sets x 12-15 reps

🔥 Day 3 – Legs & Core (Longer Session)
Quads & Glutes (Primary Focus)
1️⃣ Leg Press Machine (High/Low Foot Placement to Target Quads & Glutes) – 4 sets x 8-12 reps
2️⃣ Bulgarian Split Squats (Dumbbells or Smith Machine) – 3 sets x 10-12 reps
3️⃣ Leg Extensions Machine (Quads Isolation) – 3 sets x 12-15 reps

Hamstrings & Glutes (Secondary Focus)
4️⃣ Romanian Deadlifts (Dumbbells or Barbell) – 3 sets x 8-12 reps
5️⃣ Seated or Lying Leg Curls (Machine) – 3 sets x 12-15 reps

Calves (Support Work)
6️⃣ Calves Press Machine (Leg Press Machine with Toes on Edge) – 3 sets x 15-20 reps
7️⃣ Standing Calf Raises (Machine or Smith Machine) – 3 sets x 15-20 reps

Core (High Volume, Extra Focus Allowed)
8️⃣ Hanging Leg Raises (or Knee Raises for Beginners) – 3 sets x 12-15 reps
9️⃣ Cable or Weighted Crunches (Machine or Rope Attachment) – 3 sets x 12-15 reps
🔟 Plank (Weighted if possible) – 3 sets x 30-60 sec
